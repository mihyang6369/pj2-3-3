# 📊 하나투어 데이터 분석 프레임워크 보고서 (Analyze.md)

이 문서는 `main_2.py` 애플리케이션에 구현된 데이터 분석 내용들을 통계적 분석 단계(변수의 개수)에 따라 분류하고 그 비즈니스적 의미를 기술합니다. 본 대시보드는 초보 데이터 분석가들이 실무에서 데이터를 어떻게 구조화하는지 보여주는 가이드라인 역할을 합니다.

---

## 1. 단변량 분석 (Univariate Analysis)
*하나의 변수에 대한 기술 통계 및 분포를 확인하여 데이터의 기초 체력을 진단하는 단계입니다.*

| 분석 섹션 | 분석 내용 | 사용된 변수 | 비즈니스 의미 |
| :--- | :--- | :--- | :--- |
| **1-1** | 연도별/월별 항공 실적 | 유임승객(명) | 시장의 성장세 및 계절적 수요(시즈널리티) 파악 |
| **1-2** | 국가/도시별 누적 실적 | 유임승객(명) | 주요 거점 노선 및 시장 지배력 순위 식별 |
| **2-1** | 도시별 가격 및 평점 분포 | 성인가격, 평점 | 데이터의 편향성 및 품질 이상치(Outlier) 탐지 |
| **4-2** | 리뷰 요약 키워드 빈도 | 리뷰 키워드(태그) | 고객이 가장 빈번하게 언급하는 경험의 핵심 단어 식별 |
| **4-4** | 리뷰 등록 트렌드 추이 | 리뷰수(건) | 시계열에 따른 고객 피드백의 양적 변화 관측 |
| **4-5** | 리뷰 텍스트 길이 분포 | 리뷰 글자수 | 고객 피드백의 정성적 수준(성의도 및 분노도) 측정 |
| **5-1** | 핵심 KPI 메트릭 | 평점, 발생률 등 | 대시보드 전체의 성과를 나타내는 단일 대표 지표 |

## 2. 이변량 분석 (Bivariate Analysis)
*두 변수 간의 관계를 분석하여 특정 요인이 결과(평점, 가격 등)에 미치는 영향을 파악하는 단계입니다.*

| 분석 섹션 | 분석 내용 | 변수 조합 (X -> Y) | 비즈니스 의미 |
| :--- | :--- | :--- | :--- |
| **2-2** | 호텔 브랜드 프리미엄 | 유명브랜드여부 -> 평균가격 | 호텔 브랜드가 상품 단가 인상에 미치는 영향 검증 |
| **2-3** | 쇼핑 횟수별 상품 가격 | 쇼핑횟수 -> 성인가격 | 쇼핑 리베이트가 상품 원가 보전에 기여하는 정도 분석 |
| **3-3** | 평점 영향도 분석 | 서비스 속성 -> 영향도 | 어떤 요소가 만족도 결정에 가장 중요한지 순위화 |
| **3-4** | 인구통계 만족도 분석 | 연령/동행 -> 평점 | 타겟 고객군에 따른 상품 만족도 차이 검증 |
| **3-6** | 가격별 쇼핑 수용도 | 가격대 -> 수용도 | 가격 상승 시 쇼핑에 대한 저항이 발생하는 지점(Dead-cross) 포착 |
| **4-5** | 리뷰 길이와 평점 상관성 | 리뷰길이 -> 평점(OLS) | 불만이 클수록 리뷰가 길어지는 선형적 상관성 입증 |
| **5-3** | 시장 수요-공급 갭 분석 | 쇼핑정책 -> 수요/공급 | 시장의 니즈와 상품 공급의 불일치 구간 식별 |
| **5-5** | 전략적 타겟 LTV 분석 | 객단가 -> LTV(재구매) | 수익성(단가)과 충성도(LTV)의 상관관계를 통한 타겟팅 |

## 3. 다변량 분석 (Multivariate Analysis)
*세 개 이상의 변수를 복합적으로 고려하거나 머신러닝을 통해 숨겨진 패턴을 찾는 고도화 분석 단계입니다.*

| 분석 섹션 | 분석 내용 | 변수 조합 | 비즈니스 의미 |
| :--- | :--- | :--- | :--- |
| **1-3** | 항공 구조 및 점유율 | 도시 + 항공사 + 여객수 | 지역별 항공 경쟁 구도와 독점적 공급망 구조 분석 |
| **3-1** | 리뷰 화력 vs 만족도 | 도시 + 리뷰수 + 평점 | 양적 성과(리뷰수)와 질적 성과(평점)의 동행 여부 진단 (이중 축) |
| **3-5** | 연령x동행 평점 히트맵 | 연령대 + 동행 + 평점 | 복합 세그먼트별 만족도 집중 및 사각지대 탐색 |
| **3-6** | 도시별 쇼핑x평점 (cb3) | 도시 + 쇼핑횟수 + 평점 | 지역별 쇼핑 옵션에 대한 평점 하락 임계치 비교 분석 |
| **4-6** | 거시 버블 마켓맵 | 도시 + 쇼핑 + 평점 + 리뷰수 | 4가지 변수를 한눈에 보며 시장 포트폴리오의 위치 진단 |
| **4-7** | 상품 세그먼트 군집화 | 가격 + 평점 + 쇼핑 (K-Means) | 머신러닝을 통해 유사한 속성을 가진 상품 그룹 자동 분류 |
| **4-8** | 부정 핵심 키워드 히트맵 | 도시 + 불만 키워드 + 빈도 | 지역별 고질적 불만 원인(Pain Points)의 복합적 식별 |
| **5-8** | ROI 시뮬레이션 퍼널 | 단계 + 잔존율 + 투자액 | 구매 여정 전체 단계와 재무적 이익을 통합한 의사결정 모델링 |

## 💡 분석가들을 위한 조언
*   **단계적 접근**: 항상 단변량 분석으로 데이터의 기초 분포를 먼저 파악한 뒤, 점진적으로 변수를 늘려가며 복합적인 결론을 도출하십시오.
*   **시각화의 도구화**: 단순히 예쁜 그래프가 아니라, 위 표의 '비즈니스 의미'를 뒷받침할 수 있는 가장 명확한 차트 형태(이중 축, 히트맵, 버블맵 등)를 선택하는 것이 중요합니다.
*   **데이터 근거의 제언**: 다변량 분석의 결과(예: 4-7 군집화)는 반드시 하단의 `render_analysis_box`처럼 구체적인 액션 플랜(예: 저가 모델 구조 개선)으로 연결되어야 합니다.
